Простой плагин для создания ссылок оплаты без товаров.

== Description ==

Simple Payment Links - это простой плагин для WordPress, который позволяет создавать ссылки для оплаты произвольных сумм без необходимости создания товаров в WooCommerce.

= Основные возможности: =

* Создание ссылок для оплаты произвольных сумм
* Простая админ-панель для управления
* Интеграция с WooCommerce
* Безопасные токены для защиты ссылок
* Минимальная страница оплаты
* Поддержка всех методов оплаты WooCommerce

= Как это работает: =

1. Переходите в "Payment Links" в админ-панели
2. Указываете сумму и описание
3. Получаете ссылку для оплаты
4. Клиент переходит по ссылке и оплачивает
5. Заказ создается в WooCommerce автоматически

= Требования: =

* WordPress 5.0 или выше
* WooCommerce 5.0 или выше
* PHP 7.4 или выше

== Installation ==

1. Загрузите плагин в папку `/wp-content/plugins/simple-payment-links/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Убедитесь, что WooCommerce установлен и активирован
4. Перейдите в "Payment Links" в админ-панели для создания ссылок

== Frequently Asked Questions ==

= Нужен ли WooCommerce? =

Да, плагин требует установки и активации WooCommerce для работы.

= Можно ли использовать с любыми методами оплаты? =

Да, плагин работает со всеми методами оплаты, поддерживаемыми WooCommerce.

= Безопасны ли ссылки? =

Да, каждая ссылка защищена уникальным токеном.

= Нужно ли создавать товары? =

Нет, товары создавать не нужно. Плагин работает с произвольными суммами.
