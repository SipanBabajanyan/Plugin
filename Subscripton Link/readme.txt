Создает простые ссылки для оплаты произвольных сумм без товаров и форм checkout.

== Description ==

Subscription Link Payment - это плагин для WordPress, который позволяет создавать постоянные ссылки для ежемесячной оплаты подписки. Клиенты могут использовать одну и ту же ссылку каждый месяц для оплаты без необходимости заполнения форм checkout.

= Основные возможности: =

* Создание постоянных ссылок для подписок
* Интеграция с WooCommerce
* Минимальная страница оплаты без форм
* Автоматическое создание заказов
* Безопасные токены для защиты ссылок
* Админ-панель для управления подписками
* Статистика платежей
* Email уведомления
* Поддержка всех методов оплаты WooCommerce

= Как это работает: =

1. Создайте товар в WooCommerce и отметьте его как "Товар для подписки"
2. Перейдите в "Subscription Links" → "Create New" для создания ссылки
3. Поделитесь сгенерированной ссылкой с клиентом
4. Клиент может использовать эту ссылку каждый месяц для оплаты
5. Мониторьте платежи и статистику в админ-панели

= Требования: =

* WordPress 5.0 или выше
* WooCommerce 5.0 или выше
* PHP 7.4 или выше

== Installation ==

1. Загрузите плагин в папку `/wp-content/plugins/subscription-link/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Убедитесь, что WooCommerce установлен и активирован
4. Перейдите в "Subscription Links" в админ-панели для настройки

== Frequently Asked Questions ==

= Нужен ли WooCommerce? =

Да, плагин требует установки и активации WooCommerce для работы.

= Можно ли использовать с любыми методами оплаты? =

Да, плагин работает со всеми методами оплаты, поддерживаемыми WooCommerce.

= Безопасны ли ссылки? =

Да, каждая ссылка защищена уникальным криптографически стойким токеном.

= Можно ли изменить сумму подписки? =

Да, вы можете изменить сумму подписки через админ-панель или в настройках товара.

= Поддерживаются ли уведомления по email? =

Да, плагин поддерживает отправку уведомлений клиентам о статусе платежей.
